本週推薦

福壽山

景觀特色 : 福壽山農場佔地廣闊，四季景觀變化多端，梨、桃、長春茶皆是特產。

營業時間 : 24小時營業

地址 : 台中市和平區梨山里福壽路29號

福壽山農場位於台中市梨山南側的山坡上，四季的景觀變化多端，安排一場賞景、養生的農場旅遊。福壽山農場是設立最早，也是因此帶動整個梨山桃、梨、蘋果等高經濟作物的主要基地，不論產量、品質，福壽山農場都是箇中翹楚。

九份老街

景觀特色 : 九份上，往下可俯視基隆嶼、八斗子、番仔澳、鼻頭角等地的海景，可眺望雞籠山景。

營業時間 : 星期一 ～ 四 08:00-22:00。

地址 : 224新北市瑞芳區基山街

份老街是位於臺灣新北市瑞芳區的一條老街，主要範圍以聚集在基山街、豎崎路及輕便路等街道為主，是全台灣最熱門的觀光景點之一。若以客源分析，平日以來自日本、韓國等地的外籍觀光客以及中國大陸、香港、澳門的遊客和本國遊客各占一半，假日則以國民旅遊為主，是北台灣國際級的觀光景點。

鹿野高台

景觀特色 : 近賞整齊美麗的綠色茶園風光外，也可遠眺花東縱谷以及龍田河階上方整的農田景緻。

營業時間 : 全天

地址 : 台東縣鹿野鄉永安村高台路46號

「鹿野高台」位於台東縣鹿野鄉永安村，是龍田北側的一處高地，其特殊的地理條件，成為東部首選的優良天然空域活動場地。此區近年來最知名的活動，莫過於每年6月底至9月初的國際熱氣球嘉年華，為台灣第一個以熱氣球形象打造的人氣景點。

忘憂森林

景觀特色 : 晴天時水中有清晰的倒影、或霧濛濛時仿如夢般的景色；水窪內還有多種水生植物。

營業時間 : 早上10:00～下午4:00

地址 : 557 南投縣竹山鎮溪山路3號之25號

台灣九寨溝忘憂森林 位於南投杉林溪旁的忘憂森林，早期因921地震而形成的淤積沼澤，杉木林因長期泡水而枯死，形成了此片森林的特異景觀，腹地雖不大，但清澈的湖水倒影，彷彿來到另一個世界。 而山區午後容易起霧，霧氣瀰漫壟罩著森林，所以又稱為迷霧森林。

奇美博物館

景觀特色 : 典藏西洋藝術品為主，展出藝術、樂器、兵器與自然史四大領域。

營業時間 : 9:30～17:30（週三休）

地址 : 台南市仁德區文華路二段62號

台南奇美博物館簡介 台南奇美博物館位於台南市仁德區，靜謐的座落在台南都會公園裡，距離台南市區車程約15分鐘，相鄰保安車站。 台南奇美博物館是由許文龍先生創辦，於1992年正式開放，2014年遷移至臺南都會博物館園區奇美館，2015年重新開館，是臺灣館藏最豐最富盛名的私人博物館。

歡樂耶誕城

景觀特色 : 官方FB精選推薦「全球最令人驚豔的19棵耶誕樹 」。

營業時間 : 每年12月18:00-22:00

地址 : 220新北市板橋區縣民大道二段7號

首次舉辦是在2011年係由新北市政府與臺北市政府協議決定，新北辦耶誕，台北辦跨年，以避免互搶人潮及分散效益。由於新板特區所在位置有極為便利的四鐵共構交通樞紐，加上擁有都會中心難得的大型廣場以及周邊民間投資形塑的商圈效應，為此定型化的活動帶來了洶湧人潮，形成特定時節的標誌性活動。
